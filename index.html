Codex

WordPress Theme

View on GitHub Download .zip Download .tar.gz

Codex is a simple WordPress Themes made with "Documentations" and "Businessplans" in mind.

Features

  • Mobile-first, minimal, responsive design with very good readability
  • Integration with Download Monitor
  • Integration with Passwort Protected
  • Integration with Public Revisions
  • Integration with WP Advanced PDF and WP Advanced PDF Plus
  • Integration with Visual Composer
  • Integration with Ninja Forms
  • Integration with Polylang

Coming Soon

  • Support for Code Embedding
  • Comments
  • Rate Comments
  • Support for Posts (Blog)
  • NDA (non-disclosure-agreement; useful for online-businessplans)
